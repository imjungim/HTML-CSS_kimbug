Background

#1. background-color
#2. background-image : url('이미지 경로')
#3. background-repeat : repeat | no-repeat
#4. background-size : contain | cover | custom
contain : 안에 포함된다. 요소안에 이미지의 모든면이 다 들어갈 수 있도록 함(어쨌든 이미지가 다 들어감) .
cover : 빈공간이 남지 안도록 (빈영역을 남기지 않고 다 채움_사진짤림)
#5. background-position : x y 어떻게 위치시킬지 background-position : center center를 많이 씀.
